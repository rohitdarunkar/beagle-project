# Signup Form (HTML & CSS)

This project is a simple **Signup Form** built using **HTML, CSS, and JavaScript**.  
It collects user details and includes basic form validation with a clean, modern UI.

## Features
- Full Name, Email, Password & Confirm Password fields  
- Date of Birth input  
- Role selection (Student / Employee)  
- Terms & Conditions checkbox  
- Password match validation using JavaScript  
- Responsive and minimal dark-themed design  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (for validation)
